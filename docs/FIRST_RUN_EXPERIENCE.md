# First Run Experience

## Goal

The first run should create a meaningful 'holy shit' moment.

The goal is not to show raw profile JSON.

The goal is to demonstrate understanding.

## First Run Flow

1. Configure source.
2. Harvest artifacts.
3. Compile profile.
4. Generate first-run report.

## Deliverable 1

What Imprint Learned

Examples:

- recurring reasoning patterns
- expression differences across contexts
- notable writing shifts over time
- confidence-backed observations

## Deliverable 2

Expression Reflection

Generate a narrative summary describing the shape of the user's expression.

The output should be evidence-backed.

Not a personality assessment.

Not a diagnosis.

Not a biography.

## Deliverable 3

Expression Comparison

Show differences between:

- casual
- technical
- published
- email

where enough evidence exists.

## Deliverable 4

Profile Summary

Generate the first profile artifact.

The profile is infrastructure.

The report is the user experience.

## Anti-Goals

Do not:

- dump JSON as the primary experience
- generate personality tests
- generate diagnoses
- pretend certainty where evidence is weak
